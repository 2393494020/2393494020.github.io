#### Instruction-Set Architecture

### x86

#### 通用寄存器

- AX(AH,AL)

  Accumulator 存放乘法等指令的操作数

- BX(BH,BL)

  Base 存放存储单元的偏移地址

- CX(CH,CL)

  Count 存放计数值

- DX(DH,DL)

  Data 乘法运算产生的部分积；除法运算的部分被除数

- Remarks
  
  以上四个数据寄存器有以下特点：  
  1.均为16位寄存器  
  2.每个16位寄存器可分为两个8位寄存器使用  
  3.适用大多数算术运算和逻辑运算指令  
  4.除存放通用数据外，各有一些专门用途
